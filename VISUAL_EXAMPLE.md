## Сравнение отображения дат

### Повторяющееся занятие

**В Firestore:**
```json
{
  "dateTime": "02.03 19:00",
  "isRecurring": true
}
```

**В приложении (сегодня 15.03):**
```
📅 Пн 16.03        ← основной текст
   Пн 23.03        ← серый текст
```

---

### Ра��овое занятие

**В Firestore:**
```json
{
  "dateTime": "25.03 14:00",
  "isRecurring": false
}
```

**В приложении:**
```
📅 Ср 25.03        ← только одна дата
```

## Цветовая схема уровней

```
┌─────────────────────┐
│ Начинающий 0-1      │  ← зелёный
└─────────────────────┘

┌─────────────────────┐
│ Начинающий+ 1.5-2   │  ← зелёный
└─────────────────────┘

┌─────────────────────┐
│ Средний 2.5-3       │  ← синий
└─────────────────────┘

┌─────────────────────┐
│ Продвинутый 3-3.5   │  ← фиолетовый
└─────────────────────┘

┌─────────────────────┐
│ Продвинутый+ 4+     │  ← фиолетовый
└─────────────────────┘
```

## Особенности UX

1. **Вторая дата серым цветом** - визуально показывает, что это дополнительная информация
2. **Иконка календаря выровнена по верху** - чтобы корректно отображаться с двумя датами
3. **Hover эффект** - при наведении карточка получает синюю рамку и тень
4. **Клик на карточку** - открывает шторку с информацией о тренере
5. **Две кнопки**:
   - "Подробнее о тренере" - открывает шторку (белая с синей обводкой)
   - "Записаться" - показывает модалку с контактом (синяя)
